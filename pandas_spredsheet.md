
## `read_csv()` 
- Cargar los datos desde la fuentes externas

## `isna()` o `isnull()`
- Para detectar valores invalidos dentro del dataframe

## `dropna()`
- Limpieza directa mediante la eliminacion de registros

## `drop()`
- Eliminar etiquetas especificadas de filas o columnas.
- `drop(columns=[])` -> Elimina las columnas en especifico

## `drop_duplicates()`
- Elimina filas repititivas
- Para eliminar una especifica columna utilizar el argumento `subset`

## `replace()`
- Funcion para recodificar los valores

## Manejador `str`
### `str.strip()` o `str.replace()`
- Permite aplicar funciones de limpieza de cadenas de texto

# `astype()`
- Conversions explicita de tipos
